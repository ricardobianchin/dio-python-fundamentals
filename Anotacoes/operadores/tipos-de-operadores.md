# Tipos de Operadores

## Operadores Aritméticos
- Soma: `+`
- Subtração: `-`
- Multiplicação: `*`
- Divisão: `/`
- Divisão Inteira: `//`
- Módulo: `%`
- Potência: `**`

Exmplos:
```python
a = 10
b = 3
print(a + b)  # Soma
print(a - b)  # Subtração
print(a * b)  # Multiplicação
print(a / b)  # Divisão
print(a // b) # Divisão Inteira
print(a % b)  # Módulo
print(a ** b) # Potência
```

### precedência dos operadores aritméticos

1. Parênteses: `()`
2. Exponenciação: `**`
3. Multiplicação, Divisão, Divisão Inteira e Módulo: `*`, `/`, `//`, `%`
4. Adição e Subtração: `+`, `-`
