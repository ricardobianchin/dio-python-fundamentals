# Meu Primeiro Programa

Abra o VS Code. Pressione Ctrl + N para criar um novo arquivo. Digite o seguinte código:

```python
print("Alo, mundo!")
```

Salve o arquivo com o nome que desejar, como, por exemplo, alo_mundo.py, ou meu_primeiro_programa.py, mas sempre com a extensão obrigatória `.py`. Apesar de o uso de - como separador de palavras é aceito. O ideal é separar as palavras com underscore, que é o padrão snake_case, obrigatório para quando um script python vai executar outro. É comum utilizar o nome `main.py`. Em um programa python formado por mais de um arquivo, costuma-se dar o nome de `main.py` para o arquivo onde o programa inicia

Salve o programa pressionando Ctrl + S. Salve na pasta do seu projeto. Para executar o programa, pressione o botão de play no canto superior direito da janela do editor. Você verá a saída "Alo, mundo!" no terminal integrado do VS Code.
