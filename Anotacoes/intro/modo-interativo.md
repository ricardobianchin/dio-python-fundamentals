# modo interativo

o modo interativo do Python é uma maneira de executar código Python linha por linha, permitindo que você veja os resultados imediatamente. Ele é útil para testar pequenos trechos de código, experimentar com a linguagem e aprender os conceitos básicos.

no cmd ou terminal, você pode iniciar o modo interativo digitando `python` ou `python3`, dependendo da versão do Python instalada. uma vez no modo interativo, você verá um prompt `>>>`, onde pode digitar código Python e pressionar Enter para executá-lo.

voce pode digitar expressões, criar variáveis, definir funções e muito mais. permite que você veja os resultados imediatamente

para executar o modo interativo e já executar um scripty pyton, você pode usar o comando `python -i nome_do_arquivo.py`. isso executará o script e, em seguida, deixará você no modo interativo para continuar experimentando com o código.

para limpar a tela do modo interativo, pressione Ctrl + L 

para sair do modo interativo, você pode digitar `exit()` ou usar o atalho Ctrl + Z.
