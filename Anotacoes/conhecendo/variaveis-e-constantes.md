# variaveis e constantes

## variáveis

São um local na memória do computador onde podemos armazenar um valor de um determinado tipo

o valor colocado em uma varíavel pode ser alterado apenas indicando novo valor

o tipo da variável é definido no momento em que o valor é atribuído a ela, e pode ser alterado posteriormente

x = 10 # a variável x é do tipo inteiro
x = "Olá, Mundo!" # agora a variável x é do tipo string

## constantes

Não existem constantes em Python, mas por convenção, utilizamos letras maiúsculas no nome da constante para indicar que um valor não deve ser alterado. É apenas um acordo de cavalheiros, pois nada nos impede de alterar o valor da constante

PI = 3.14159 # constante para o valor de pi
GRAVIDADE = 9.81 # constante para a gravidade na Terra
DEBUG = True # constante para indicar se o modo de depuração está ativado

## o que é snake_case?

snake_case é um estilo de nomenclatura onde as palavras são separadas por underscores (_). É comumente utilizado para nomear variáveis, funções e arquivos em Python.

exemplos aceitos: soma, media_ponderada, maior_valor, MAXIMO_PERMITIDO
exemplos não aceitos: soma Total, mediaPonderada, MaiorValor, MAXIMOPERMITIDO

## boas práticas para nomear variáveis e constantes

- Use nomes descritivos e significativos para as variáveis e constantes, para que seja fácil entender o propósito do valor armazenado.
- Evite usar nomes muito curtos ou genéricos, como "x" ou "data", a menos que seja claro no contexto.
- Use letras minúsculas para variáveis e letras maiúsculas para constantes, seguindo a convenção do snake_case.
